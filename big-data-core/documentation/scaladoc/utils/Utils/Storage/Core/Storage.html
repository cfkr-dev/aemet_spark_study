Storage

case class Storage(storagePrefix: Option[String], s3Endpoint: Option[String] = None) extends Product with Serializable

Abstraction over storage backends that can handle both local filesystem paths and s3:// URIs. The storagePrefix is prepended to requested paths and an optional s3Endpoint can be provided for testing.

storagePrefix

optional prefix to prepend to paths (throws if missing)

s3Endpoint

optional S3 endpoint for testing/localstack
