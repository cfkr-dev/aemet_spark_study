  1.  object ChronoUtils

    Utility methods for simple timing and delays.

    Utility methods for simple timing and delays.

    Provides helpers to execute blocks with a minimum elapsed time and a lightweight Chronometer for measuring and formatting elapsed time.

    Example:

    val chrono = ChronoUtils.Chronometer()
chrono.start()
// do work
val elapsed = chrono.stop() // returns formatted `String` like "00:01:23.456"
